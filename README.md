# Init Project

Basic bootstrap project to build applications with grunt

Basic setup: https://www.smashingmagazine.com/2013/10/get-up-running-grunt/
Deploys to: http://dev.jonbrownm.com/init-project/

## Includes

- SASS compliation
- Javascript bundling
- Localhost creation
- Watch tasks
- FTP push

## Development task
`grunt`

## Deployment task
`grunt prod`
